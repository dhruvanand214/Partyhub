**Dusk Till Dawn** is a premium, AI-powered SaaS application designed for the vibrant Indian nightlife. Whether you're planning a massive house party, looking for the perfect solo vibe, or need a "cure" for the next morning, we've got you covered.

---

## ✨ Features

### 🥂 Smart Party Planner
Planning a desi party shouldn't be a headache. 
- Input your guest count, drink preferences, and budget.
- Get precise quantities (Bottles, Mixers, Snacks).
- AI-generated food pairings and playlist vibes.

### 🥃 Solo Vibe Mode
Just you and the night?
- Tell us your mood (Chill, Hype, Melancholy).
- Get personalized drink recipes and pairing suggestions.
- Discover new labels that fit your specific vibe.

### 💊 Hangover Recovery (The Cure)
Kal ki party, aaj ka dard? 😅
- Tell us your symptoms and hydration levels.
- Get a scientifically-backed recovery plan.
- Recommended foods, hydration schedules, and rest tips.

### 📅 Dry Day Alert
Never get caught off guard.
- Real-time tracking of Dry Days across major Indian cities.
- Get notified before the shops close.

## 🚀 Tech Stack

- **Framework:** [Next.js 15 (App Router)](https://nextjs.org)
- **Styling:** [Tailwind CSS](https://tailwindcss.com) & [Framer Motion](https://www.framer.com/motion/)
- **Authentication:** [Clerk](https://clerk.com)
- **Database:** [MongoDB](https://www.mongodb.com) with Mongoose
- **AI Engine:** [OpenRouter AI](https://openrouter.ai/) (GPT-4 / Claude)
- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Design System:** Custom "Afterglow Futurism" UI

## 💰 SaaS & Freemium Model

Dusk Till Dawn operates on a sustainable Freemium model:
- **Free Tier:** 3 AI-generated plans per month.
- **Dawn Pro:** Unlimited generations, history access, and priority support.
- **Paywall Integration:** Seamlessly integrated with Razorpay for instant upgrades.

## 🛠️ Local Setup

1. **Clone the repo:**
   ```bash
   git clone https://github.com/yourusername/partyhub.git
   cd partyhub
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file and add:
   ```env
   MONGODB_URI=your_mongodb_uri
   OPENROUTER_API_KEY=your_key
   NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_key
   CLERK_SECRET_KEY=your_key
   CLERK_WEBHOOK_SECRET=your_key
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```
